# Backend Tenko — REST API (Node.js)

## Overview
Backend REST API developed for a client web project,
built with Node.js. Handles server-side logic, routing,
and data management.

## Tech Stack
- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** MySQL
- **Architecture:** REST API

## Features
- User authentication
- API routing and middleware
- Server-side data processing
- Client-server communication
